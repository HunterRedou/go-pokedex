With this Application you can play a Text based Pokemon. It calls a Pokemon API.
The Help Command will show you all Command that aviable to you.
Welcome to the Pokedex!
Usage:

catch: Catch a Pokemon
inspect: Inspect the Info over Catched Pokemon
pokedex: Shows Pokedex with your caught Pokemon
exit: Exit the Pokedex
help: Displays a help message
map: Shows Area Locations
mapb: Goes back to the Previous Locations
explore: Explores the Area Location

Have Fun with it.
